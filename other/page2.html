Vue.js Tutorial

Setup and Installation

There are two main ways to set up Vue - in a Node project, or directly injected into a static HTML file. Let's see how to set up Vue in an HTML file, as it's the simplest setup and introduction.


Steps

  1. As a prerequiste: Install Node:
    For instructions follow this npm installation tutorial
  2. Open command prompt and type:
    # install with npm
    npm i -g @vue/cli @vue/cli-service-global
  3. Congratulations you have installed vue
